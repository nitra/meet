# 3. Медіа та пристрої

## Пре-джойн (PreJoin)

Перед входом у кімнату користувач бачить екран PreJoin, де задає:

- **Ім’я учасника** (username)
- Увімкнення **відео** (за замовчуванням увімкнено)
- Увімкнення **аудіо** (за замовчуванням увімкнено)

Ці значення передаються як `LocalUserChoices` і далі використовуються при публікації треків у кімнаті.

## Меню налаштувань (Settings)

Меню налаштувань (`SettingsMenu`) доступне в кімнаті (якщо увімкнено через `NEXT_PUBLIC_SHOW_SETTINGS_MENU`). Воно об’єднує:

- **Медіа-пристрої** — камера, мікрофон, динамік (перемикачі та вибір пристроїв)
- **Запис** — кнопка старт/стоп запису (якщо налаштовано `NEXT_PUBLIC_LK_RECORD_ENDPOINT`)

Підменю медіа побудовані на компонентах **CameraSettings** та **MicrophoneSettings**, які дають вибір пристроїв і перемикання треків (камера/мікрофон).

## Перемикання треків у кімнаті

- **TrackToggle** (з LiveKit Components) — кнопки вмикання/вимкнення камери та мікрофона в UI.
- **Швидкі клавіші** (розділ «Користувацький досвід»): Cmd/Ctrl+Shift+A — мікрофон, Cmd/Ctrl+Shift+V — камера.

## Кодеки та якість

- На сторінці кімнати підтримуються параметри **codec** (наприклад, vp9) та **hq** (вища якість).
- Для кастомного підключення кодек можна передати в query (`codec=vp9` тощо); підтримуються лише валідні значення з `livekit-client` (наприклад, vp8, vp9, av1).

## Оптимізація

- Використовується **useLowCPUOptimizer** для зниження навантаження на CPU в кімнаті (наприклад, зменшення роздільності при нестачі ресурсів).
